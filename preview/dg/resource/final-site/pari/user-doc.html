Pari v-0-0-1 User Docs

Hi, Welcome to the core branch of pari.

This doc is solely writtent to explain you, How you can use pari to ease you life and write beautiful css without actually writing it.

Let's have a look at main ingredients of Pari -
  • Pari's theme file system to provide you extreme customization power
  • Pari's semanti-element system that gives you liberty to make you pages just by typing normal html tags with minimal classes
  • Pari's helper classes provide you an upped edge and gives you freedom to make you desired layout with few classes
Folder structure of pari

Now let's have a look at folder structure of Pari -

beauty - Its the main folder where you will get css of - components, elements, helpers, layout, thems, and core pari.css file

assets - images or illustration used for making sample pages

html - This folder contins the html for Pari's elements and components

Now Let's see how can you make pages for you apps using pari and its customizable themes

Suppose You are making a overview page for you webapp and you need a Navbar to as you first componet of page

You can just simply go to the "html/navbar.html " and can copy paste the html for the navbar from there and can paste it to you html file

By defualt you will get a simple navbar like which you are seeing above

You can cutomize it as per you need by modifying its theme file which you will get at "beauty\themes\light\components\navbar-theme-pari.css"

For e.g. We have changed the horontal aligment of navbar here to center 

  
           CSS
           
            --navbar-horizontal-alignment: center;

and got something alike this

So In a similar way you can modify any css property from nabar theme file and can get you cutomized navbar
